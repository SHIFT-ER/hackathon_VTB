# PostgreSQL Data Migration CLI

CLI-утилита для миграции данных между базами данных PostgreSQL. Утилита поддерживает выборочную миграцию данных, работу с различными форматами конфигурации (YAML, JSON), а также многопоточность.

## Структура проекта

- **`bin/`** — Скрипты для запуска.
- **`config/`** — Примеры конфигурационных файлов (YAML/JSON).
- **`src/`** — Исходный код миграции.
- **`tests/`** — Тесты для функционала.
- **`logs/`** — Логи выполнения миграции.
